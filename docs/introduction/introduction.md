Raf-Core is library belong to project [Autofiy](https://www.github.com/autofiy). this library help developers 
creating forms in very customizable manner. when you use this library you will have ready to go:

* UI (Rendering)
* Validation
* Submitting
* Collecting
* Events


### Installation

using **npm**

    npm install @autofiy/raf-core

using **yarn**

    yarn add @autofiy/raf-core
